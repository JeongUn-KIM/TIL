# Markdown

> text를 html로 변환하는 언어
>
> 최대한 읽기 쉽게 최소한의 문법으로 구조화

---

## 제목(Heading)

+ 문서의 제목이나 소제목으로 사용함

+ 문서의 구조를 위한 것이며 **글자 크기 조절로 사용 X** 

  > # h1
  >
  > ## h2
  >
  > ### h3
  >
  > #### h4
  >
  > ##### h5
  >
  > ###### h6

​	

***

## 목록(List)

+ 항목을 나열할 때 사용

+ 순서 有 : <ol><li>

+ 순서 無 : <ul><li>

> 근데 그냥 워드에 할 때처럼 +, - 사용해서 리스트업 해도
>
> 아래와 같이 점으로 통일돼서 출력됨

<ol> 순서가 있는 리스트
    <li>First</li>
    <li>Second</li>
    <li>Third</li>
</ol>

<ul> 순서가 없는 리스트
    <li>이거</li>
    <li>저거</li>
    <li>그거</li>
</ul>
